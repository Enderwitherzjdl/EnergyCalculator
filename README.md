# Energy Converter (能量单位转换器)

一个使用 **PyQt6** 编写的桌面应用，用于在常见物理、光谱、化学能量单位之间快速转换。支持 **eV、nm、cm⁻¹、THz、fs、ps、K、J、kJ、kcal/mol、kJ/mol** 等多种单位，界面简洁，带数字键盘与单位键盘，可自动格式化科研常用指数表示。

---

## ✨ 功能特点

* 支持多种能量及相关物理量的快速互转
* 单位包括：
  **eV, nm, cm⁻¹, THz, ps, fs, K, J, kJ, kcal/mol, kJ/mol**
* 输入数字可使用内置数字键盘
* 单位选择按钮自动高亮
* 输出格式自动对齐并支持科学计数法（指数使用 Unicode 上标）
* 输出内容整体居中显示、可调行距
* 采用模块化结构：`run.py` + `energy_core.py` + `main_gui.py`

## 🚀 安装与运行

### 1. 安装依赖

安装 PyQt6：

```bash
pip install PyQt6
```

### 2. 运行程序

```bash
python run.py
```

在 Windows 下编译并打包成程序：
```bash
./CompilePackage.bat
```

---

## 📘 使用说明

1. 输入框中输入数值，可使用数字键盘辅助输入
2. 点击一个单位按钮（如 `eV` 或 `cm-1`）
3. 程序将自动计算并在右侧文本框显示所有单位的对应值
4. 单位按钮会高亮显示当前选中的单位
5. 输出文本自动居中并等宽对齐，便于阅读和复制
